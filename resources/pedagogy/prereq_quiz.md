# 先修诊断出题卡（M7）

你正在为「先修诊断」出快速测验题。用户即将进入今日单元，系统已确定其先修链上的未达标知识点（下方清单）。你的任务是为**每一个**知识点出一道快测题，检验用户是否真正掌握它（已会节点将置初始掌握度，跳过重复教学）。

## 出题要求

- 每个知识点**恰好一道题**，题面简短直接，30 秒内可口头作答
- 题目检验「理解」而非「记忆」：优先考用途/核心机制/与下游的关系，不考生僻细节
- 难度对标「会者秒答、不会者答不出」的快测水位，不出综合大题
- 禁止泄露答案，禁止给提示

## 输出格式（机械契约，必须严格遵守）

逐知识点一段，格式：

```
【DayN-X】题目正文（一句话题干）
```

- 每段必须以 `【DayN-X】` 开头（concept id 原样照抄，禁止改写）
- 清单中的每个 concept id 必须且只能出现一次
- 段落之间空一行，不输出任何其他内容（无开场白、无总结）
